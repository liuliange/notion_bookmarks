# 置顶卡片下方新增「标签广告栏」功能教程

## 一、功能概述

在置顶推广卡片（带 `置顶推广` 标签的大卡片）下方，新增一行**胶囊式标签广告栏**，用于展示次要推广内容。

所有推广内容在「导航链接」数据库中统一管理，通过打 `标签广告` 标签控制显示。

**视觉层级：**

```
┌─────────────────────────────────────────┐
│  置顶推广卡片 (md+, 最多 5 个)            │
│  [LinkCard] [LinkCard] [LinkCard] ...    │
├─────────────────────────────────────────┤
│  标签广告栏 (md+)       ← 本次新增        │
│  [💎 设计] [🎬 视频] [💻 代码] ...       │
├─────────────────────────────────────────┤
│  小组件 (lg+)                             │
│  [时钟] [天气] [热搜] ...                 │
└─────────────────────────────────────────┘
```

## 二、技术方案

| 项目 | 方案 |
|------|------|
| **数据来源** | 「导航链接」数据库，通过 `标签广告` 标签过滤 |
| **API 端点** | 新建 `/api/tag-ads`，独立于 `/api/promoted-links` |
| **标签形态** | 胶囊 pill（`rounded-full`），图标 + 标题，最多显示 6 个字 |
| **交互方式** | 点击跳转（`cursor-pointer`手指）；空白区拖拽滑动（`cursor-grab`手掌） |
| **拖拽防误触** | 位移 > 5px 视为滑动，`pointer-events-none` 阻止点击 |
| **展示设备** | 仅桌面端 `hidden md:block`，移动端隐藏 |
| **无数据兜底** | 无 `标签广告` 数据时整个区域不渲染 |

## 四、Notion 配置步骤

### 4.1 给记录打标签

在「导航链接」数据库中，找到需要推广的记录，在 `Tags`（多选）字段中添加选项值 **`标签广告`**。

```
导航链接数据库（已存在）
├── 记录 A → Tags: 标签广告
├── 记录 B → Tags: 标签广告
├── 记录 C → Tags: 标签广告, 力荐👍
└── ...
```

> 注意：一个记录可以同时拥有 `置顶推广` 和 `标签广告` 两个标签，互不冲突。带有 `置顶推广` 的会显示为大卡片，带有 `标签广告` 的会显示为标签 pill。

### 4.2 无需额外配置

标签广告栏**不需要**在「网站配置」数据库中额外配置。有数据就显示，无数据就隐藏。

## 五、代码详解

### 5.1 API 接口：`src/app/api/tag-ads/route.ts`

```typescript
import { NextResponse } from 'next/server';
import { getLinks } from '@/lib/notion';

export async function GET() {
    try {
        const allLinks = await getLinks();

        const tagAdLinks = allLinks.filter(link => {
            if (Array.isArray(link.tags)) {
                return link.tags.includes('标签广告');
            }
            if (typeof link.tags === 'string') {
                return link.tags.includes('标签广告');
            }
            return false;
        });

        return NextResponse.json({
            success: true,
            links: tagAdLinks
        });
    } catch (error) {
        console.error('获取标签广告失败:', error);
        return NextResponse.json(
            { success: false, error: '获取数据失败' },
            { status: 500 }
        );
    }
}
```

**关键点：**
- 复用 `getLinks()`（已有 React `cache()` 去重，不会额外消耗 Notion API 请求配额）
- 不限制返回数量
- 不过滤标签（保留所有标签用于展示）

### 5.2 胶囊标签组件：`src/components/ui/TagAdPill.tsx`

```tsx
'use client';

import { Link } from '@/types';
import { cn } from '@/lib/utils';

interface TagAdPillProps {
    link: Link;
    className?: string;
    theme?: string;
}

export default function TagAdPill({ link, className, theme }: TagAdPillProps) {
    // 根据主题获取颜色
    const getThemeColors = () => {
        if (theme?.includes('cyberpunk')) {
            return { bg: 'bg-[hsl(286,100%,65%)]', text: 'text-white' };
        }
        if (theme?.includes('bauhaus')) {
            return { bg: 'bg-[var(--bauhaus-red)]', text: 'text-white' };
        }
        if (theme?.includes('macintosh')) {
            return { bg: 'bg-[var(--mac-ink)]', text: 'text-[var(--mac-paper)]' };
        }
        // simple-light / simple-dark 及其他默认主题
        return { bg: 'bg-primary', text: 'text-primary-foreground' };
    };

    const colors = getThemeColors();

    return (
        <a
            href={link.url}
            target="_blank"
            rel="noopener noreferrer"
            draggable={false}
            className={cn(
                "flex items-center gap-1.5 rounded-full px-3 py-1.5 shrink-0 cursor-pointer",
                colors.bg,
                colors.text,
                className
            )}
        >
            {link.iconfile && (
                <img
                    src={link.iconfile}
                    alt=""
                    draggable={false}
                    className="w-4 h-4 rounded object-contain"
                    onError={(e) => (e.target as HTMLImageElement).style.display = 'none'}
                />
            )}
            <span className={cn("text-sm font-medium truncate max-w-[90px]", colors.text)}>
                {link.name}
            </span>
        </a>
    );
}
```

**关键点：**
- `<span>` 同时设置 `colors.text`，防止赛博朋克主题 `a:hover` 规则覆盖文字颜色（显式声明优先于继承）
- `max-w-[90px]` + `truncate`：6 个中文字完整显示，第 7 个字起 `...`
- `draggable={false}`：防止浏览器原生拖拽产生阴影
- `cursor-pointer`：手指样式，告诉用户可点击

**主题颜色适配：**

| 主题 | 标签背景 | 标签文字 |
|------|----------|----------|
| 简约浅色 | `bg-primary` | `text-primary-foreground` |
| 简约深色 | `bg-primary` | `text-primary-foreground` |
| 赛博朋克 | `bg-[hsl(286,100%,65%)]` | `text-white` |
| 包豪斯 | `bg-[var(--bauhaus-red)]` | `text-white` |
| 麦金塔 | `bg-[var(--mac-ink)]` | `text-[var(--mac-paper)]` |

### 5.3 标签栏容器：`src/components/ui/TagAdBar.tsx`

```tsx
'use client';

import { useRef, useState, useCallback, useEffect } from 'react';
import { Link } from '@/types';
import { cn } from '@/lib/utils';
import TagAdPill from './TagAdPill';

interface TagAdBarProps {
    links: Link[];
    theme?: string;
}

const DRAG_THRESHOLD = 5;

export default function TagAdBar({ links, theme }: TagAdBarProps) {
    const scrollRef = useRef<HTMLDivElement>(null);
    const mouseDown = useRef(false);
    const startX = useRef(0);
    const scrollStart = useRef(0);
    const [isDragging, setIsDragging] = useState(false);

    // 全局 mouseup 清理：防止拖拽到容器外释放鼠标导致状态残留
    useEffect(() => {
        const handleGlobalMouseUp = () => {
            mouseDown.current = false;
            setIsDragging(false);
        };
        window.addEventListener('mouseup', handleGlobalMouseUp);
        return () => window.removeEventListener('mouseup', handleGlobalMouseUp);
    }, []);

    const onMouseDown = useCallback((e: React.MouseEvent) => {
        mouseDown.current = true;
        startX.current = e.pageX;
        scrollStart.current = scrollRef.current?.scrollLeft ?? 0;
        setIsDragging(false);
    }, []);

    const onMouseUp = useCallback(() => {
        mouseDown.current = false;
        setIsDragging(false);
    }, []);

    const onMouseMove = useCallback((e: React.MouseEvent) => {
        if (!mouseDown.current) return;
        e.preventDefault();
        const dx = e.pageX - startX.current;
        // 位移超过阈值视为拖拽，阻止标签点击跳转
        if (Math.abs(dx) > DRAG_THRESHOLD) {
            setIsDragging(true);
        }
        if (scrollRef.current) {
            scrollRef.current.scrollLeft = scrollStart.current - dx;
        }
    }, []);

    const onMouseLeave = useCallback(() => {
        mouseDown.current = false;
        setIsDragging(false);
    }, []);

    if (links.length === 0) return null;

    return (
        <div
            ref={scrollRef}
            className={cn(
                "w-full overflow-x-auto scrollbar-none select-none [-webkit-user-drag:none]",
                isDragging ? "cursor-grabbing" : "cursor-grab"
            )}
            onMouseDown={onMouseDown}
            onMouseUp={onMouseUp}
            onMouseMove={onMouseMove}
            onMouseLeave={onMouseLeave}
        >
            <div className={cn(
                "flex items-center gap-3",
                isDragging && "pointer-events-none"
            )}>
                {links.map((link) => (
                    <TagAdPill key={link.id} link={link} theme={theme} />
                ))}
            </div>
        </div>
    );
}
```

**关键点：**

| 机制 | 实现 |
|------|------|
| **位移阈值** | `DRAG_THRESHOLD = 5`，位移 ≤ 5px 视为点击，> 5px 视为拖拽 |
| **防误点击** | 拖拽中内层容器加 `pointer-events-none`，标签 `<a>` 完全收不到 click |
| **全局清理** | `window.addEventListener('mouseup')`，拖到容器外释放也能正常恢复 |
| **拖拽阴影** | `[-webkit-user-drag:none]` + 所有 `<a>` 和 `<img>` 设置 `draggable={false}` |
| **鼠标样式** | 标签上 `cursor-pointer`（手指=可点击）；空白区 `cursor-grab`（手掌=可拖拽） |
| **Hooks 规则** | `if (links.length === 0) return null` 放在所有 hooks 之后 |

### 5.4 页面组件：`src/components/widgets/HomeWidgets.tsx`

在推广卡片区和小组件区之间插入标签广告栏渲染块。

**新增内容：**

```tsx
// 1. 导入标签栏组件（第 10 行）
import TagAdBar from '@/components/ui/TagAdBar';

// 2. 新增状态（第 75-77 行）
const [tagAdLinks, setTagAdLinks] = useState<Link[]>([]);
const [loadingTagAds, setLoadingTagAds] = useState(true);

// 3. 新增数据获取（第 97-115 行）
useEffect(() => {
    let cancelled = false;
    fetch('/api/tag-ads')
        .then((res) => res.json())
        .then((data) => {
            if (!cancelled && data.success && data.links) {
                setTagAdLinks(data.links);
            }
        })
        .catch(() => {
            if (!cancelled) setTagAdLinks([]);
        })
        .finally(() => {
            if (!cancelled) setLoadingTagAds(false);
        });
    return () => {
        cancelled = true;
    };
}, []);

// 4. 显隐判断（第 118 行）
const hasTagAds = !loadingTagAds && tagAdLinks.length > 0;

// 5. 渲染区块（第 156-167 行，推广卡片与小组件之间）
{hasTagAds && (
    <div className="hidden md:block w-full mb-8 px-4">
        <motion.div
            initial={{ opacity: 0, y: -10 }}
            animate={{ opacity: 1, y: 0 }}
            transition={{ duration: 0.5, delay: 0.1 }}
        >
            <TagAdBar links={tagAdLinks} theme={theme} />
        </motion.div>
    </div>
)}
```

**间距说明：**

| 区域边界 | 间距值 | 含义 |
|----------|--------|------|
| 置顶卡片上方（页面顶部） | `pt-4` | 16px |
| 置顶卡片 → 标签广告栏 | `mb-2` | 8px |
| 标签广告栏 → 小组件 | `mb-8` | 32px |

## 七、踩坑记录

### 7.1 写入文件内容为空导致 "Element type is invalid"

并行写入多个文件时 `TagAdPill.tsx` 内容未成功保存，导致 `TagAdBar` 导入时拿到空模块。

**解决**：重新单独写入 `TagAdPill.tsx`。

### 7.2 赛博朋克主题悬停时文字不可见

赛博朋克主题 CSS 有全局规则 `[data-theme="cyberpunk-dark"] a:hover { color: hsl(286,100%,65%) }`，覆盖了标签 `<a>` 的 `text-white`。标签背景正好是 `hsl(286,100%,65%)`，文字与背景同色。

**解决**：将 `colors.text` 同时加到内层 `<span>`，使其拥有显式颜色声明（显式 > 继承）。

### 7.3 点击与拖拽容易混淆

最初的实现按下一个像素就进入拖拽状态，导致每次点击都会触发滑动。

**解决**：引入 5px 位移阈值。移动 ≤ 5px = 点击，> 5px = 拖拽 + `pointer-events-none` 阻止 click。

### 7.4 拖拽到容器外释放鼠标后状态不恢复

`onMouseLeave` 可以处理鼠标离开容器，但如果用户在容器外释放（例如拖到浏览器边缘），`mouseup` 不会触发组件上的事件。

**解决**：使用 `window.addEventListener('mouseup', ...)` 注册全局监听，确保任何位置释放鼠标都能清理拖拽状态。

### 7.5 拖拽时产生浏览器原生拖拽阴影（ghost）

浏览器会默认拖拽 `<img>` 和带 `href` 的 `<a>`，产生半透明阴影。

**解决**：
- 外层容器加 `[-webkit-user-drag:none]` CSS
- `<a>` 标签加 `draggable={false}`
- `<img>` 标签加 `draggable={false}`
